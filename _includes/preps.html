{% highlight python %} class Solution: def addStrings(self, num1: str, num2: str) -> str: sum, carry = '', 0 i, j = len(num1) - 1, len(num2) - 1 while i >=0 or j >= 0: x = ord(num1[i]) - ord('0') if i >= 0 else 0 y = ord(num2[j])- ord('0') if j >= 0 else 0 sum += str((x+y+carry)%10) carry = (x+y+carry)//10 j -=1 i -=1 if carry > 0: sum += '1' return sum[::-1] ''' Steps: - right to left scan each character and add using `ord() - ord('0')` - keep adding their sum's rightmost digit (%10) to `sum`, and left (should be 0 or 1) to carry - reverse the final `sum` after appending carry to it (because you were appending left to right) ''' {% endhighlight %}
